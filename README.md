# @khanshamshad32/carousel

A react native library.


## Install

`$ npm install @khanshamshad32/carousel`


## Usage

<CircularCarousel
        dataSource={dataSource}
        onItemPress={item => {
          console.log(item);
        }}
        containerDim={{height: 200, width: 350}}
        itemDim={{width: 100, height: 110}}
        radius={100}
      />
